Technical Architecture

Microservices architecture designed for security, scalability, and compliance

Technology Stack

Frontend

  • iOS: Swift, SwiftUI
  • Android: Kotlin, Jetpack Compose
  • PWA: React, Next.js
  • Admin: React, TypeScript

Backend

  • API Gateway: Kong or AWS API Gateway
  • Services: Node.js (TypeScript) or Go
  • Database: PostgreSQL 14+
  • Cache: Redis
  • Queue: RabbitMQ

Infrastructure

  • Cloud: AWS or GCP
  • Storage: S3/Cloud Storage
  • Monitoring: Datadog/New Relic
  • CI/CD: GitHub Actions

Integrations

  • Payments: Stripe, Plaid
  • KYC: Jumio/Onfido
  • Blockchain: Chainalysis
  • Notifications: Firebase, Twilio
